The created record is returned in the response.

{@inheritdoc}

{@request:json_api}
Example:

```JSON
{
  "data": {
    "type": "customerusers",
    "attributes": {
      "confirmed": true,
      "email": "AmandaFCole@example.org",
      "firstName": "Amanda",     
      "lastName": "Cole",     
      "enabled": true,
      "password": "Password000!"
    },
    "relationships": {
      "roles": {
        "data": [
          {
            "type": "customeruserroles",
            "id": "1"
          }
        ]
      },
      "customer": {
        "data": {
          "type": "customers",
          "id": "1"
        }
      }
    }
  }
}
```
{@/request}
